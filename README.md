## Fonctionnalités

- Aiguilles animées en continu (50 ms) — heures, minutes, trotteuse rouge
- Affichage digital avec heures, minutes et secondes
- Date complète en français — jour, numéro, mois
- Bascule **mode clair / mode sombre** avec transition fluide
- Texture de grain subtile en fond
- Typographies [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) et [DM Mono](https://fonts.google.com/specimen/DM+Mono)
- Sans dépendance, zéro framework

---

## Utilisation

Ouvre `Horloge/index.html` directement dans un navigateur — aucune installation requise.
